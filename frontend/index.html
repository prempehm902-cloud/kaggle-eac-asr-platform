Notebooks and APIs

Code / API Integration for notebooks, mobile clients, backend services, and external tools using copyable endpoints and Python examples.

POST /api/v1/transcriptions

Upload one audio file and receive language, transcript, confidence, and segments.

POST /api/v1/transcriptions/batch

Send multiple audio files and return JSONL/CSV-ready batch output.

POST /api/v1/feedback

Submit corrected transcripts for the human feedback loop.

GET /api/v1/lab/leaderboard

Read per-language WER scores for Swahili, Kikuyu, Luo, Somali, Maasai, and Kalenjin.

Python client example

import requests

with open("audio.wav", "rb") as audio:
    response = requests.post(
        "http://localhost:8000/api/v1/transcriptions",
        files={"file": audio},
        data={"detect_language": "true", "return_segments": "true"},
    )

print(response.json())
