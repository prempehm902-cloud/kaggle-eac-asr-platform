# Team and Leaderboard Attestation

This project tracks the AfriVoice EAC ASR hackathon constraints:

- Team size must be 5 participants or fewer.
- The team must use one leaderboard account.
- The Kaggle username should be provided separately from this application profile URL.

Before final submission, update this file with the official Kaggle team name, Kaggle username, and final participant list.
