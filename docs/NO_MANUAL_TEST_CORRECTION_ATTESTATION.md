# No Manual Test Correction Attestation

The Kaggle test-set prediction workflow for this project must remain fully automated.

- Test audio must not be manually transcribed.
- Test predictions must not be manually corrected by humans.
- Human review and transcript correction are allowed only for training, validation, development, or user-owned audio records.
- Final Kaggle `submission.csv` files must be generated by model inference code.

This file exists so the competition validation workflow can verify that the project documents and enforces the no-manual-correction rule.
