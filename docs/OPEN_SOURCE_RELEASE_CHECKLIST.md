# Open-Source Release Checklist

Before final leaderboard submission, publish the following under an OSI-approved permissive license such as MIT, Apache-2.0, BSD-3-Clause, or MPL-2.0:

- source code
- training scripts
- inference scripts
- trained/fine-tuned checkpoints or weights
- model card
- data card references provided by organizers
- training logs and reproducibility notes
- hardware validation report

The repository already includes an MIT license. Real model weights still need to be published and referenced in `models/checkpoints/model-metadata.json`.
